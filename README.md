# 📊 Visual Intuition

## 🔥 Neural Network
```
Input → Hidden Layers → Output
```

## 🌳 Ensemble (Random Forest)
```
Tree1 + Tree2 + Tree3 → Final Prediction
```

## 📍 KNN
```
Find nearest points → Majority vote
```

## 🎯 SVM
```
Maximize margin between classes
```

## ⚡ Gradient Boosting
```
Model1 → Error → Model2 → Improve
```

## 🧪 CNN
```
Image → Filters → Feature Maps
```

## 🔁 Reinforcement Learning
```
State → Action → Reward → Learn
```

## 🔍 PCA
```
High Dim → Reduced Dim (max variance)
```

# 📐 Key Formulas

## Logistic Regression
$$
P(y=1|x) = \frac{1}{1 + e^{-(wx + b)}}
$$

## KNN Distance
$$
d(x, x_i) = \sqrt{\sum (x - x_i)^2}
$$

## SVM
$$
w \cdot x + b = 0
$$

## Gradient Boosting
$$
F_m(x) = F_{m-1}(x) + \gamma h_m(x)
$$

## CNN
$$
(I * K)(x,y) = \sum I(x+i, y+j)K(i,j)
$$

## GANs
$$
\min_G \max_D V(D,G)
$$

## Reinforcement Learning
$$
Q(s,a) = r + \gamma \max Q(s',a')
$$

## Transformers
$$
Attention(Q,K,V) = softmax\left(\frac{QK^T}{\sqrt{d_k}}\right)V
$$

# 🧩 Pros & Cons Snapshot

| Algorithm | Strength | Weakness |
|----------|---------|---------|
| DNN | Powerful | Black box |
| Ensemble | Robust | Complex |
| Logistic | Interpretable | Limited |
| KNN | Simple | Slow |
| SVM | Precise | Not scalable |
| Boosting | High accuracy | Overfit risk |
| CNN | Best for images | Data heavy |
| Linear | Fast | Too simple |
| Naive Bayes | Efficient | Assumptions |
| Autoencoder | Feature learning | Hard to explain |
| GAN | Realistic data | Hard training |
| RL | Adaptive | Expensive |
| Transfer | Efficient | Depends on base |
| Transformer | Context aware | Heavy |
| PCA | Dimensionality reduction | Info loss |
